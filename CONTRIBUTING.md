This repository aims to be a comprehensive collection of resources on Medical World Models, including disease progression simulation, patient trajectory prediction, treatment planning, and clinical decision support.

## How to Contribute

### Adding a New Paper

1. **Fork** this repository
2. **Clone** your fork locally
3. **Add** the paper following the format below
4. **Commit** your changes with a clear message
5. **Push** to your fork
6. **Submit** a Pull Request

### Paper Format

Please use the following format when adding papers:

```markdown
- **[Paper Title](arxiv/doi link)** [Mon., Year]  
  Author1, Author2, Author3, et al.  
  *Venue, Year* | [[PDF]](link) [[Code]](link) [[Project]](link)
```

**Example:**

```markdown
- **[Medical World Model: Generative Simulation of Tumor Evolution for Treatment Planning](https://arxiv.org/abs/2506.02327)** [Jun., 2025]  
  Yijun Yang, et al.  
  *arXiv, 2025* | [[PDF]](https://arxiv.org/abs/2506.02327) [[Project]](https://yijun-yang.github.io/MeWM/)
```

### Paper Categories

Please add papers to the appropriate category:

| Category | Description |
|----------|-------------|
| **Survey Papers** | Review papers covering medical world models or related topics |
| **Disease Progression Simulation** | Papers on simulating disease dynamics over time |
| **Patient Trajectory Prediction** | Papers on predicting patient health trajectories |
| **Treatment Response Modeling** | Papers on modeling treatment responses |
| **Patient Digital Twins** | Papers on creating digital patient representations |
| **Reinforcement Learning for Healthcare** | Papers applying RL to healthcare |
| **Dynamic Treatment Regimes** | Papers on adaptive treatment strategies |
| **Medical Image Synthesis** | Papers on generating synthetic medical images |
| **Evaluation & Benchmarks** | Benchmarks and evaluation methods |
| **Datasets** | Public datasets for medical world model research |

### Guidelines

- Papers should be peer-reviewed or from reputable preprint servers (arXiv, medRxiv, bioRxiv)
- Include all available links (PDF, Code, Project page)
- Place papers in **reverse chronological order** within each section (newest first)
- Use consistent formatting
- Add a brief description (1-2 sentences) for significant papers using `>` blockquote
- Do not include papers without accessible links
- Do not include duplicate entries
- Do not include papers unrelated to medical world models

### Adding a New Category

If you think a new category should be added:

1. Open an **Issue** first to discuss
2. Propose the category name and description
3. List at least 3 papers that would belong to this category
4. Wait for maintainer approval before submitting PR

### Adding a Dataset

When adding a new dataset, please use the following table format:

```markdown
| Dataset Name | Description | Size | Modality | Link |
|--------------|-------------|------|----------|------|
```

### Reporting Issues

- **Broken links**: Please open an issue with the paper title and broken link
- **Incorrect information**: Please open an issue with the correction details
- **Suggestions**: Feel free to open an issue for any suggestions

## Pull Request Process

1. Ensure your PR has a clear title and description
2. Reference any related issues
3. Wait for review from maintainers
4. Address any feedback promptly
5. Once approved, your PR will be merged

### Commit Message Format

```
Add: [Paper Title] to [Category]
Fix: Broken link for [Paper Title]
Update: [Section Name] with new resources
```

## Code of Conduct

- Be respectful and inclusive
- Provide constructive feedback
- Focus on the content, not the person
- Help maintain a welcoming environment for all contributors

## Scope of This Repository

This repository focuses on **Medical World Models**, which includes:

- Disease progression simulation
- Patient trajectory prediction
- Treatment response modeling
